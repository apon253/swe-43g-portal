
# SWE 43-G | Academic Portal

Welcome to the official academic portal for **Section SWE 43-G** at Daffodil International University (DIU). As the Class Representative (CR), I developed this centralized hub to make academic life easier and more organized for my classmates. 

This platform eliminates the need to constantly scroll through WhatsApp groups to find class links, drive folders, or routine updates. Everything is right here in one place!

### Key Features
* **Real-time Notice Board:** Instant announcements pushed directly from a secure Admin Panel using Firebase Firestore. No page refreshes needed!
* **Course Hub:** One-click access to specific Google Drive folders for recorded classes, materials, notes, and BLC links for every ongoing course.
* **Dynamic Schedules:** Visual display of current Class Routines, Lab Schedules, and Exam updates.
* **Dark Mode Magic:** Fully functional dark mode for those late-night study sessions.
* **Responsive Design:** Looks great and works flawlessly on both mobile and desktop.

### Tech Stack Used
* **Frontend:** HTML5, CSS3 (Custom Variables & Animations), Vanilla JavaScript (ES6+).
* **Backend/Database:** Google Firebase (Firestore Database).
* **Hosting:** Netlify / GitHub Pages.

### Why I Built This
Being a Junior Penetration Tester and a SWE student, I believe in building tools that solve real-world problems. Managing links and notices for a whole section was getting chaotic, so I engineered this solution to keep everyone synced and productive.
